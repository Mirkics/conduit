Conduit blog applikáció:

Az applikáció fő funkcióinak listája a https://github.com/Mirkics/conduit/blob/master/docs/functions.txt fileban található

Az applikáció tesztjeinek lokális futtatásához a szükséges környezetet telepíteni kell.
Ennek listája megtalálható a https://github.com/Mirkics/conduit/blob/master/requirements.txt

A github a futtatáskor a conduit_app.yml filet használja
https://github.com/Mirkics/conduit/blob/master/.github/workflows/conduit_app.yml

A vizsgához az applikáció manuális felület tesztjeinek listája elérhető:
https://github.com/Mirkics/conduit/blob/master/docs/Conduit_Testcases.xls

Ezek alapján lehet kiválasztani, hogy mely teszteseteket érdemes automatizálni.
Ez alapján az alábbi funkciók lettek automatizálva:
https://github.com/Mirkics/conduit/tree/master/pytests

A testsuit nem teljes. Az elvárt eredmény alapján csak a pozitív végkimenetre készültek teszesetek eddig.
Vannak teszesetek, amelyben több funkció tesztelése történik.

A github-on van lehetőség az eredményekről készített riportot megtekinteni.
Allure report: https://mirkics.github.io/conduit/
